Detailed Description

XDDP (Cross Domain Datagram Protocol) pipes: useful to communicate beetween N-RT and RT threads.

It can be used both in Xenomai and standard linux kernel: XDDP in the former case, FIFO in the latter. Check https://xenomai.org/documentation/trunk/html/api/xddp-echo_8c-example.html for more info.
